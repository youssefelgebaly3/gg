# MotoLock - تطبيق التحكم في الدراجة النارية

تطبيق Flutter للتحكم في الدراجة النارية عبر البلوتوث مع إمكانية التحكم الصوتي.

## المميزات

### التحكم الأساسي
- **قفل/فتح** الدراجة
- **تشغيل/إيقاف** المحرك
- **تنبيه** صوتي
- **مارش** للبدء

### التحكم الصوتي
- **"ذكري شغلي الموتسيكل"** أو **"دوري الموتسيكل"** → فتح → تشغيل → مارش (7 ثوان)
- **"أقفل الموتسيكل"** → قفل
- **"أوقف المحرك"** → إيقاف

### مميزات إضافية
- إعادة الاتصال التلقائي
- حفظ إعدادات الجهاز
- واجهة RTL للعربية
- دعم الثيم المظلم/الفاتح
- رسائل تنبيه ذكية

## التثبيت

1. تأكد من تثبيت Flutter SDK
2. استنسخ المشروع
3. شغل الأوامر التالية:

```bash
flutter pub get
flutter run
```

## الصلاحيات المطلوبة

- **البلوتوث**: للاتصال بالأجهزة
- **الموقع**: مطلوب للبحث عن أجهزة البلوتوث
- **الميكروفون**: للتحكم الصوتي

## الاستخدام

### الاتصال
1. فعّل البلوتوث
2. اضغط "بحث" للعثور على الأجهزة
3. اختر الجهاز من القائمة
4. اضغط "اتصال"

### التحكم الصوتي
1. فعّل "التحكم الصوتي" من الإعدادات
2. اضغط على أيقونة الميكروفون
3. قل أحد الأوامر:
   - "ذكري شغلي الموتسيكل"
   - "دوري الموتسيكل"
   - "أقفل الموتسيكل"
   - "أوقف المحرك"

### الأزرار اليدوية
- **قفل**: قفل الدراجة
- **فتح**: فتح القفل
- **تشغيل**: تشغيل المحرك
- **إيقاف**: إيقاف المحرك
- **تنبيه**: تفعيل التنبيه
- **مارش**: بدء المحرك (يحتاج تشغيل أولاً)

## المكتبات المستخدمة

- `flutter_blue_plus`: للتحكم في البلوتوث
- `permission_handler`: لإدارة الصلاحيات
- `shared_preferences`: لحفظ الإعدادات
- `audioplayers`: لتشغيل الأصوات
- `speech_to_text`: للتحكم الصوتي

## ملاحظات

- التطبيق يعمل محلياً بدون إنترنت
- التحكم الصوتي يدعم اللغة العربية
- يتم حفظ حالة الأزرار تلقائياً
- إعادة الاتصال التلقائي عند فتح التطبيق
