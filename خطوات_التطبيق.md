# خطوات تطبيق MotoLock

## المشكلة:
الملف الحالي تالف ويحتوي على أخطاء. 

## الحل:
استخدم الملفات النظيفة الجديدة.

## الخطوات:

### 1. انسخ الملفات النظيفة:
- انسخ محتوى `main_clean.dart` → `lib/main.dart`
- انسخ محتوى `pubspec_clean.yaml` → `pubspec.yaml`

### 2. احذف الملف القديم:
- احذف `lib/main.dart` القديم
- احذف `pubspec.yaml` القديم

### 3. أعد تسمية الملفات:
- `main_clean.dart` → `main.dart`
- `pubspec_clean.yaml` → `pubspec.yaml`

### 4. تشغيل التطبيق:
```bash
flutter pub get
flutter run
```
