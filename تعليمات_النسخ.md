# تعليمات نسخ الكود

## الملفات المطلوبة:

### 1. ملف main.dart
- انسخ محتوى ملف `main_complete.dart` 
- ضعه في مجلد `lib/main.dart` في مشروع Flutter

### 2. ملف pubspec.yaml  
- انسخ محتوى ملف `pubspec_complete.yaml`
- ضعه في مجلد المشروع كـ `pubspec.yaml`

## خطوات التطبيق:

1. **إنشاء مشروع Flutter جديد:**
   ```bash
   flutter create moto_lock_app
   cd moto_lock_app
   ```

2. **استبدال الملفات:**
   - انسخ محتوى `main_complete.dart` إلى `lib/main.dart`
   - انسخ محتوى `pubspec_complete.yaml` إلى `pubspec.yaml`

3. **تثبيت المكتبات:**
   ```bash
   flutter pub get
   ```

4. **تشغيل التطبيق:**
   ```bash
   flutter run
   ```

## المميزات:

### 🎤 التحكم الصوتي:
- **"ذكري شغلي الموتسيكل"** أو **"دوري الموتسيكل"** → فتح → تشغيل → مارش (7 ثوان)
- **"أقفل الموتسيكل"** → قفل
- **"أوقف المحرك"** → إيقاف

### 🔗 إدارة الاتصال:
- البحث عن أجهزة البلوتوث
- الاتصال/قطع الاتصال
- إعادة الاتصال التلقائي
- حفظ عنوان MAC

### 🎮 الأزرار:
- قفل/فتح، تشغيل/إيقاف، تنبيه، مارش
- منع العمليات غير المسموحة
